1. Discuss the scope of var, let, and const

Var

Var can declare many times with same name. In var we can reassign values also can re declare.

Let

We can declare let one time with a name. But we can reassign values in let.

Const

Const declarations like let one time with a name, but we can't re assign values in a const variable.

2. Tell us the use cases of null and undefined

If we want to declare variable but the value is not decided yet, then we can set null. And if we don't assign null then the variable value is set undefined by default.
